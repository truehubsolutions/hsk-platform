# Contributing

This is a private client platform maintained by [True Hub Solutions](https://truehubsolutions.com). The repository is public for portfolio purposes — the documentation reflects real architecture and outcomes.

## This repo contains documentation only

No production source code is stored here. The actual codebase, credentials, and client data are maintained in private infrastructure.

## Feedback & suggestions

If you're a developer with ideas for improving the architecture or documentation:

1. Open an issue using the [feature request template](.github/ISSUE_TEMPLATE/feature_request.md)
2. Describe the improvement and which module it affects
3. Pull requests improving documentation clarity are welcome
